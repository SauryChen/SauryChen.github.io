Download Remote Sensing Data

From Google Earth Engine

We use download Sentinel-1 Descending orbit, IW Mode, VV polarization from Google Earth Engine (GEE) as an example. For other widely-used remote sensing data such as Landsat series, operations are similar.

After that, you can visualize the images.

I am not sure about the range of min and max values here, here is a useful website: What are the min and max values of Map.addLayer on Google Earth Engine.

From Sentinel Website with Python API

You can also download Sentinel-1 images from the Copernicus Open Access Hub. You can either use the graphical interfaces or the Python API sentinelsat. Here is a detailed tutorial

Install sentinelsat in terminal with pip install sentinelsat
