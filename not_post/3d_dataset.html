ScanNet 原始数据集

  • ScanNet对于原点存在偏移，数据处理参考3D点云处理偏移

  • .sens格式读取：提供C++和python版本，Python需要使用2.7。

  • .ply数据读取：使用meshlab。

scannet_frames_25k.zip

由于原始ScanNet数据量特别大，作者提供了较小子集的选项scannet_frames_25k.zip以及基准评估scannet_frames_test。

  • Label (uint8): 每个像素储存nyu40id对应的值

  • Instance (int32): instance = label * 1000 + inst。inst为图片中每个label的第几个实例。0表示未标注或没有深度。
