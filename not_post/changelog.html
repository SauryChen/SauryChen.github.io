ChangeLog

Ever Since I built this website, I have modified it for many times, from the themes to small widgets, although not many contents are added. However, after changing many themes, I found Maupassant is really simple but elegent. Therefore, I decided to gradually improve my blog based on Maupassant, and hope not to blow hot and cold anymore.

📧 Gitalk or Valine?

I continue to use Gitalk after registering LeanCloud, which is a one-stop back-end service (?) Valine based on. LeanCloud needs real-name authentication, which I don’t like. Besides, Valine is not safe, (although I don’t evaluate the safety of gitalk). For detailed explanation, refer to https://imnerd.org/valine-may-not-so-good.html.

🔍 Google Search Console

Several trails are made when submitting my personal blog to Google Search Console. Follow these steps can ensure a fluent submission but some pages remain unfound by Google search engine.

  1. Go to Google Search Console to varify the website, use https://<YOUR_NAME>.github.io/

  2. Download the html and submit to the root directory of hexo. For example, you can find the <HTML_NAME>.html on https://<YOUR_NAME>.github.io/<HTML_NAME>.html. However, if you fail at this step, submit the html directly via GitHub upload file instead of using Hexo. Since Hexo submission will change the file. Then varify your website.
